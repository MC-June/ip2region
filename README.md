# ip2region

#### 介绍

提供国家，省，市，运营商papi变量

| 变量名                         | 显示内容 | 示例        |
|-----------------------------|------|-----------|
| %ip2region_region%          | 总区域  | 中国上海上海市电信 | 
| %ip2region_national%        | 国家   | 中国        | 
| %ip2region_provincial%      | 省    | 上海        | 
| %ip2region_municipal%       | 市    | 上海市       | 
| %ip2region_serviceProvider% | 运营商  | 电信        |
